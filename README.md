JSDigestAuth
============

Digest Authentication using Spring Security and JavaScript

Requirements:
 - Java 6 or greather
 - Maven 3 or greather
 
How to run:
 - Open console (cmd in Windows)
 - Navigate to project directory
 - Execute:
    maven tomcat:run
 - Open browser 
 - Open url: http://localhost:8080/restauth/resources/html/test.html
