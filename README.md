JSDigestAuth
============

Digest Authentication using Spring Security and JavaScript

Requirements:
 - Java 6 or greather
 - Maven 3 or greather
 
How to run:
1) Open console (cmd in Windows)
2) Navigate to project directory
3) Execute:
   maven tomcat:run
4) Open browser 
5) Open url: http://localhost:8080/restauth/resources/html/test.html
